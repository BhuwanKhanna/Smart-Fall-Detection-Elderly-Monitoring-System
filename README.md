The system is developed using a robust AI-driven tech stack centered around Python, which serves as the core programming language for implementing real-time logic and model integration. OpenCV is used for video stream handling, frame extraction, and image preprocessing, enabling continuous monitoring through a camera feed. For human pose detection, the project utilizes MediaPipe Pose, which provides precise body keypoint tracking to analyze posture and detect falls or abnormal movements. Data processing and pattern analysis are supported by NumPy and Pandas, allowing efficient computation and logging of activity data. For backend functionality and system interaction, lightweight frameworks such as Flask or FastAPI can be incorporated to manage alerts, APIs, and data flow. The visualization layer can be built using Streamlit or basic web technologies (HTML, CSS, JavaScript) to display incident logs and monitoring dashboards. This integrated stack enables a scalable, real-time, and intelligent elderly monitoring solution combining computer vision and practical deployment capabilities.
